# Deroom AI — AI Room Design from a Photo

[Deroom AI](https://deroomai.com) is a photo-based **AI room design** tool. Upload a photo of your real room, and the model returns a redesigned version of the same space — same walls, windows, and floor plan, with different finishes, colors, or furniture.

## How it differs from text-to-image models

Generic image generators take a text prompt and produce *a* room. The walls move. The window ends up somewhere else. The output is beautiful but it isn't your room.

Photo-based [AI room design](https://deroomai.com) uses your photo as a structural constraint via depth and segmentation conditioning. Geometry stays locked while style varies — useful for deciding whether to repaint cabinets sage green or off-white before you spend the money.

## What you can do

- **Recolor only** — material colors change, geometry holds
- **Quick refresh** — keep the bones, swap finishes
- **Declutter** — strip everything except structure (real-estate agents use this for listing prep)
- **Furniture swap** — replace one piece in place
